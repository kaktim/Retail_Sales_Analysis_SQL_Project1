# Retail Sales SQL Analysis Project

## Project Overview

This project analyzes retail sales transactions using SQL Server Management Studio (SSMS) and Tableau to generate business insights and visualize sales performance.

The project demonstrates practical SQL skills required for Data Analyst, BI Analyst, and Junior Data Engineer roles, including:

- Data Cleaning
- Exploratory Data Analysis
- Business Intelligence Reporting
- Advanced SQL Querying
- Dashboard Development

---

# Tools & Technologies

- SQL Server Management Studio (SSMS)
- SQL Server
- Tableau
- GitHub
- CSV Dataset

---

# Dataset Information

Dataset: `SQL_Retail_Sales_Analysis_utf.csv`

The dataset contains retail transaction records including:

| Column Name | Description |
|---|---|
| transactions_id | Unique transaction ID |
| sale_date | Transaction date |
| sale_time | Transaction time |
| customer_id | Customer identifier |
| gender | Customer gender |
| age | Customer age |
| category | Product category |
| quantity | Quantity purchased |
| price_per_unit | Product unit price |
| cogs | Cost of goods sold |
| total_sale | Total sales value |
